
# Technical Movies App

## Tech Stack

- Angular 14 | Javascript - Typescript
- Tailwinds | CSS - SCSS

## Features

- RXJS

## Run Locally

Clone the project

```bash
  git clone https://github.com/andreverach/technical-movies-app.git
```

Go to the project directory and install dependencies

```bash
  npm install
```

Start the server

```bash
  ng serve -o
```
